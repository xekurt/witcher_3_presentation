# Maps in The Witcher 3: Wild Hunt

A presentation exploring the spatial design, atmosphere, and mechanics of the three main regions in *The Witcher 3: Wild Hunt*: Velen, Novigrad, and Skellige.

## Overview

This project is a web-based presentation built with [Reveal.js](https://revealjs.com/). It covers:
- **Velen**: The war-torn "No Man's Land".
- **Novigrad**: The bustling "Free City".
- **Skellige**: The rugged archipelago.

## Technologies Used
- HTML5
- CSS3 (Custom styling with Witcher-themed variables)
- Reveal.js (Presentation framework)

## How to View
Open `index.html` in your web browser.
